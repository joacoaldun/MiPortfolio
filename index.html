Conocimientos

Juego de dados:Gran Cerdo

Realizado para la materia Laboratorio 1, consiste en un juego de dados que se juega entre 2 jugadores, 1 contra el otro, donde a través de diferentes reglas se van sumando puntos, el que acaba con más puntos es el ganador. Realizado completamente en el lenguaje C++, con Programación Estructurada, mostrando todo por consola.

Sistema para gestionar reservas para hoteles

Realizado para la materia Laboratorio 2, consiste en un sistema que nos permite gestionar todo lo relacionado a reservas de habitaciones para hoteles. Realizado completamente en el lenguaje C++, con Programación Orientada a Objetos, haciendo uso de archivos como base de datos, y mostrando todo por consola.

Sistema de gestión de artículos

Realizado para la materia Programación 3, consiste en un sistema para poder listar, buscar, agregar, modificar y eliminar artículos(y las imágenes que estos poseen, puede ser 1 sola o varias), también podemos hacer lo mismo con marcas y categorías. Realizado en el lenguaje C#, usando Programación Orientada a Objetos Y modelo de dominio, Microsoft SQL Server como Base de datos, en .NET Framework.
